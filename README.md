# Nom du projet
Task Tracker (Gestionnaire de tâches)

## Description du projet
Task tracker est un projet utilisé pour suivre et gérer vos tâches. Dans ce projet, vous créez une interface de ligne de commande (CLI) simple pour suivre ce que vous devez faire, ce que vous avez fait et ce sur quoi vous travaillez actuellement. Ce projet vous aide à mettre en pratique vos compétences en programmation, notamment en travaillant avec le système de fichiers, en gérant les entrées utilisateur et en créant une application CLI simple.

## Installation
git clone https://github.com/Dodou12/Task_Tracker.git
npm install

## Usages
Pour lancer le projet `npm start` ou `node index.js`

## Structure des fichiers
`index.js` : C'est le point d'entrée de mon application
`handler.js` : il réunit toutes les fonctions utilisées dans `index.js`
`tasks.json` : C'est le fichier généré quand on lance le projet pour la première fois, il contient les tâches enregistrées

## Technologies
Node.js, JSON, readline

## Fonctionnalités
CRUD sur une tâche
Recherche une tâche
Filtrer les tâches
